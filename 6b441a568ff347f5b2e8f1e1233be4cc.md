Port Numbers

<ins>Common port numbers</ins>

- SMB port is 139 (NETBios), 445
- SMTP is port 25
- FTP 20,21
- SimpleFTP - 115
- NTP - 123
- SNMP - 161
- Telnet - 23
- RDP - 3389
- DHCP - 67,68
- DNS - 53
- IMAP - 143
- POP3 - 110
- RPCBind - 111 \[Do NFS Enumeration\]
- NFS - 2049 usually (find RPCBind will find nfs)
